# CoCodes landing page

Marketing site for [CoCodes](https://github.com/VinceZcrikl/CoCodes) — Next.js 15 + Tailwind CSS v4, bilingual (EN/中文).

## Develop

```sh
npm install
npm run dev   # http://localhost:3000
```

## Deploy to Vercel

Import this repository in Vercel and set **Root Directory** to `landing`. The framework preset (Next.js) is detected automatically — no other configuration needed.
